## TypeScript

#### Basic Types
```js
// Boolean
let isDone: boolean = false;

// Number
let decimal: number = 6;

// String
let color: string = "blue";

// Array
let list: number[] = [1, 2, 3];
let list: Array<number> = [1, 2, 3]; // Same as the first approach, but uses a generic array type

// Tuple
// A tuple is an array type where the exact position and their types is specified
let props: [string, number];


```
