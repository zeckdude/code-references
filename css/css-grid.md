## CSS Grid

#### Syntax
| Keyword       | Description                                                     |
|---------------|-----------------------------------------------------------------|
| `display: grid` | Sets the element's contents as the columns of the row |

#### Create a basic grid
##### [display](https://developer.mozilla.org/en-US/docs/Web/CSS/display)
```html
<div class="container">
  <div>1</div>
  <div>2</div>
  <div>3</div>
  <div>4</div>
  <div>5</div>
  <div>6</div>
</div>
```

```css
.container {
  display: grid;
}
```
