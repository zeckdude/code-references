## Data Types

#### Get the datatype for the specified value or variable
##### [List of data types](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Data_structures#Data_types)
```js
typeof { course: "daytona" }
// Returns: "object"

typeof "hello there"
// Returns: "string"

var num = 2;
typeof num;
// Returns: "number"

typeof true;
// Returns: "boolean"
```
